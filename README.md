# thibautRe's dotfiles

Collection of dotfiles

## Installation

```sh
mkdir ~/git && cd ~/git && git clone git@github.com:thibautRe/dotfiles.git
cd dotfiles && ./install/fedora.sh
```

## Contents

### Utilities

- kitty (terminal)
- helix (editor)
- zsh
  - oh-my-zsh
	- pure (prompt)
